OpenESP (Open Enterprise Search Platform) is a powerful enterprise search server based on Apache™ Solr™.

The product includes:
* Apache Solr 6
* Apache Tomcat 8
* Apache ManifoldCF
* Apache ZooKeeper 3
* SolrMeter
* ViFun
* OpenESP administration page

NOTE: We recommend 64 bit OS and Java

The product is open source, released under the Apache Software License.
Read more at http://www.openesp.org/
